# public

Static assets served by Next.js as-is.

## Rules

- Files are available from root URL (`public/x` -> `/x`).
- Do not put business logic or generated build files here.
- Keep naming predictable (`icons/`, `images/`, `illustrations/`).

## Examples

- `public/logo.svg` -> `/logo.svg`
- `public/icons/menu.svg` -> `/icons/menu.svg`
- `public/images/login-bg.jpg` -> `/images/login-bg.jpg`
